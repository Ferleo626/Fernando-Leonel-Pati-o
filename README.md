# Sitio Web Informativo

Proyecto de una página web estática desarrollada como práctica personal.

## Descripción
Sitio web informativo con estructura básica, enfocado en aplicar buenas prácticas de maquetado y diseño web.

## Tecnologías utilizadas
- HTML5
- CSS3

## Características
- Estructura semántica
- Estilos personalizados
- Diseño responsive básico

## Objetivo
Practicar el desarrollo de sitios web estáticos y afianzar conocimientos de HTML y CSS.
# Ejercicios de Lógica en JavaScript

Repositorio con ejercicios prácticos de lógica de programación.

## Descripción
Conjunto de ejercicios realizados para practicar fundamentos de JavaScript y lógica computacional.

## Tecnologías utilizadas
- JavaScript

## Contenidos
- Variables y tipos de datos
- Condicionales
- Bucles (while, do while)
- Funciones
- Ejercicios de lógica básica

## Objetivo
Desarrollar pensamiento lógico y mejorar la comprensión de los fundamentos de la programación.
