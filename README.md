# Sitio Web Informativo

Proyecto de una página web estática desarrollada como práctica personal.

## Descripción
Sitio web informativo con estructura básica, enfocado en aplicar buenas prácticas de maquetado y diseño web.

## Tecnologías utilizadas
- HTML5
- CSS3

## Características
- Estructura semántica
- Estilos personalizados
- Diseño responsive básico

## Objetivo
Practicar el desarrollo de sitios web estáticos y afianzar conocimientos de HTML y CSS.
